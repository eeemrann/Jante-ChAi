
# Jante ChAI

**Jante ChAI** is an AI-powered web platform aimed at helping citizens of Bangladesh access and understand government services such as NID, passport, birth registration, and more. The project uses a chatbot-based interface to provide clear, step-by-step guidance in both Bengali and English.

---

## 🚧 Project Status: In Development

This project is currently in active development. The MVP (Minimum Viable Product) will focus on:
- A responsive web interface
- Chatbot assistance for NID-related queries
- A basic search and retrieval system for common government services

---

## 🧠 Core Features (Planned)

- 🤖 **Chatbot Assistance**  
  Get instant help through a friendly AI chatbot on NID-related services.

- 🔍 **Search and Retrieval System**  
  Search for instructions, forms, and service locations using plain language.

- 🌐 **Bilingual Support**  
  Interface available in both Bengali and English.

- 📱 **Mobile-Responsive Design**  
  Fully responsive layout using Bootstrap to ensure compatibility across devices.

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6+), Bootstrap 5  
- **Backend (Planned):** None (for MVP; may use Firebase or a custom Node.js server later)  
- **AI/NLP (Planned):** Integration with OpenAI API or a custom rule-based response system
