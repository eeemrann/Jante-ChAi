# Jante ChAI

**Jante ChAI** is an AI-powered web platform that helps citizens of Bangladesh understand and access government services like NID, passport, birth registration, and more. The system uses a smart chatbot interface powered by the **Grok API** to provide real-time, accurate assistance in Bengali and English.

---

## 🚧 Project Status: In Development

The project is currently under active development. The Minimum Viable Product (MVP) will feature:
- A responsive Bootstrap-based web interface
- Chatbot assistance for NID-related services
- Basic query handling using the Grok API
- Bilingual interface (Bangla & English)

---

## 🧠 Core Features (Planned)

- 🤖 **Grok-Powered Chatbot**  
  Seamless chatbot experience using Grok API for understanding and answering user queries.

- 🔍 **Search and Retrieval System**  
  Plain-language search functionality for government forms, instructions, and procedures.

- 🌐 **Bilingual Interface**  
  Supports both Bengali and English language queries and responses.

- 📱 **Mobile-Responsive Design**  
  Built with Bootstrap to look great on all devices.

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6+), Bootstrap 5  
- **AI/NLP:** Grok API (via secure token access)  
- **Backend (Planned):** None for MVP, but optional Node.js or Firebase later  
- **Deployment (Planned):** GitHub Pages or Netlify
